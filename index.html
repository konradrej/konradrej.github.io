RCPC

What started out as a simple client app for a university course about app development turned into a personal project that lives on. Since handing in the client app I have properly implemented a server as well as added the nearby servers feature with the help of multicast DNS. Some minor things such as adding a dark mode and more has also been done at the time of writing.

At this point (9th of January 2022) I have lots more planned for the whole project. At the minimum I want to add a keyboard view to the client, I also want to rewrite the communication between the server and client as I have come to learn that the architecture I picked was not great. But these are the things you learn working on actual software and not essentially demos that you hand in at university.
